# Check your setup

Everything runs on your Mac — the checklist is short:

- **macOS 26 (Tahoe) or later** on **Apple Silicon**
- **Apple Intelligence enabled** — System Settings → Apple Intelligence & Siri
- **Bridge CLI** — on macOS 27+ the `fm` CLI is preinstalled (nothing to do);
  on macOS 26, install the fallback: `brew install scouzi1966/afm/afm`

Run **Apple Foundation Models: Run Setup Check** from the Command Palette to verify all of
this automatically. If something is missing you'll get a button pointing at the fix.
