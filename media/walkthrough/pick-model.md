# Pick the on-device model

1. Open the **Chat** view (`⌃⌘I`).
2. Click the **model picker** at the bottom of the chat input.
3. Choose **Apple On-Device**.

The first request starts the local bridge automatically — you'll see the sparkle in the
status bar once it's healthy. From here on, every token is generated on your Neural Engine:
no API key, no account, and it works with Wi-Fi off.
